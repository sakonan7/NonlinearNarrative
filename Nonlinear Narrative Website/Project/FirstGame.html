Mystic Bard

"Final Fantasy III Battle Scene 1" By Nobuo Uematsu

In the class CDT 208, I've continued learning how to make miscellaneous digital art through Adobe Creative Suite. Learning about the Processing program, a program that uses Java programming to generate art, I decided to make my first game. I made a game titled Mystic Bard, an app style, beat 'em up game involving using instruments to destroy monsters and defend a village.
